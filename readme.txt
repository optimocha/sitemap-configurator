A tiny plugin to configure the WordPress core sitemap.
 
== Description ==
 
Since WordPress 5.5, we have sitemaps in the core. Sitemap Configurator lets you have more control over the core sitemap.
 
== Installation ==
 
1. Search for "sitemap configurator" in the Plugins > Add New screen.
1. Install and activate the plugin.
1. Configure it via the Sitemap Configurator page under Settings.
